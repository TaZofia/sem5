Aby zainstalować jedyną potrzebną zewnętrzną bibliotekę należy w REPL wpisać:
import Pkg
a następnie:
Pkg.add("DataStructures")

Program uruchamia się poleceniem:
julia main.jl 
w katalogu głównym projektu.
Należy dodać odpowiednie flagi tak jak w poleceniu do zadania.
